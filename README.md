# OMEGA-Monitoring-IoT_kelompok1-CDP500-MAnagement-energy
OMEGA adalah sistem monitoring panel listrik berbasis IoT yang memantau tegangan, arus, daya, suhu, dan kelembaban secara real-time. Dengan Power Meter PM810MG, sensor DHT11, dan ESP32, data dikirim melalui MQTT ke dashboard web untuk memudahkan teknisi memantau kondisi panel dan mendeteksi potensi gangguan lebih cepat.
